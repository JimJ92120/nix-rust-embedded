# nix-rust-embedded

Toolchains and environments for embedded development with `rust` and `nix`.

---

# toolchains

- [`aarch64-none`]: example targets **Raspberry Pi 3B** (`BCM2837`)
- [`arm-none`]: example targets **Raspberry Pi 2B v1.1** (`BCM2836`)
- [`avr`]: example targets **Arduino Uno** (`ATmega328p`)
